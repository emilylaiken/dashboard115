The next step is to ensure that the call log is in the proper format for upload. While the copy originally downloaded from Verboice ought to suffice, it is worth the time to ensure that the following columns are included in the log. While the columns need not appear in the same order they are in below, they must have the same spelling and capitalization. The dashboard will automatically detect which public and HC diseases are present in a call log, as long as column names follow the conventions listed below. A template of an uploadable CSV file can be downloaded here.

General hotline-related columns
  • ID
  • Started
  • Caller interaction
  • Duration(second)
  • Caller ID
  • Status
Public hotline-related columns
  • hotline_menu
  • disease_menu
  • A column for each public disease's menu, named as disease_menu
HC report-related columns
  • level_worker
  • cdc_report_started
  • cdc_report_ended
  • Two columns for each HC disease, named as var_disease_case and var_disease_death
